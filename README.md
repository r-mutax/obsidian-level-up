## ðŸ“¥ Installation

This plugin is currently **not** available in the official Obsidian Community Plugins list. You need to install it manually.

1.  Go to the **[Releases](https://github.com/r-mutax/obsidian-level-up/releases)** page of this repository.
2.  Download the latest release assets:
    - `main.js`
    - `manifest.json`
    - `styles.css`
3.  Open your Obsidian Vault folder.
4.  Navigate to `.obsidian/plugins/` (If `.obsidian` is hidden, you may need to enable hidden files).
5.  Create a new folder named `obsidian-level-up`.
6.  Move the 3 downloaded files (`main.js`, `manifest.json`, `styles.css`) into this folder.
7.  Restart Obsidian (or reload plugins).
8.  Go to **Settings > Community Plugins** and enable **Obsidian Level Up**.

## ðŸŽ® How to Use

Just use Obsidian as usual!

- **Link**: Connecting notes gives you XP bonuses.
- **Create**: New notes grant significant XP.
- **Streak**: Open Obsidian every day to keep your streak alive and earn daily bonuses.

### Dashboard
Click the **Chart Icon** in the left ribbon to open your **Level Up Dashboard**.
